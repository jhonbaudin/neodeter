# Asset Library

## Estructura actual

- `brand/`
  - identidad visual oficial
- `company/team/`
  - equipo y retratos corporativos
- `company/events/`
  - ferias, stands y presencia comercial
- `company/visits/`
  - visitas a planta, clientes o recorridos tecnicos
- `sectors/laundry/`
  - lavanderia industrial y comercial
- `sectors/food/`
  - industria alimentaria e inocuidad
- `sectors/horeca/`
  - hoteles, restaurantes y cocinas
- `sectors/maintenance/`
  - mantenimiento industrial
- `sectors/agroindustry/`
  - agroindustria
- `sectors/healthcare/`
  - hospitales y limpieza especializada

## Nota

Esta fase reorganiza los assets ya disponibles localmente en una libreria mas clara.
Las imagenes adjuntadas en la conversacion que no se encontraron como archivos locales no se mezclaron en esta estructura para evitar cargar referencias ambiguas.
